Browser Extension Installation

Current Version: TiltGuard v1.1.0 (Feb 26, 2026)

Supported Casinos: Stake.com, Stake.us, Roobet, BC.Game, Duelbits, Rollbit, Shuffle, Gamdom + generic support

Chrome Installation

  1. Download tiltcheck-extension.zip from GitHub repository
  2. Unzip the file to a local folder
  3. Open Chrome and navigate to chrome://extensions/
  4. Enable "Developer mode" (toggle in top-right)
  5. Click "Load unpacked" and select the unzipped folder
  6. Navigate to any supported casino site ‚Äî the extension will activate automatically

Firefox Installation

  1. Download browser-extension.zip from GitHub repository
  2. Open Firefox and navigate to about:debugging#/runtime/this-firefox
  3. Click "Load Temporary Add-on"
  4. Select the manifest.json file from the unzipped folder

Provably Fair Verification

For casinos that use provably fair systems (like Stake, Roobet, BC.Game), you can upload your bet history and cryptographic seeds to verify fairness.

What You Can Upload

  • CSV files: Export from casino (bet ID, wager, payout, seed, nonce)
  • Server seeds: Unhashed seeds for cryptographic verification
  • Client seeds: Your custom seeds if you set them

How to Export from Stake

  1. Log in to Stake.com or Stake.us
  2. Go to Settings ‚Üí Bet History
  3. Click "Export" and select CSV format
  4. Upload to TiltCheck dashboard
  5. We'll verify each bet cryptographically and flag anomalies

Privacy Note: All verification happens locally in your browser. Seeds and bet data never leave your device.

Frequently Asked Questions

Is the extension safe?

Yes. The extension is open-source and only reads casino data ‚Äî it never writes or modifies anything. All code is available on GitHub for review.

Does it work on all casinos?

The extension has site-specific selectors for major casinos (Stake, Roobet, BC.Game, etc.) but also includes generic selectors that work on many other sites. If your casino isn't supported, let us know and we'll add it.

Will casinos detect or ban me for using this?

No. The extension only reads public data that's already visible in your browser ‚Äî it doesn't modify anything, bot, or automate gameplay. It's like taking notes on paper, just automated.

Can I use this on mobile?

The Chrome extension requires desktop Chrome/Firefox, but the PWA sidebar works on any mobile device. Just add the TiltCheck dashboard to your home screen.

Does it track me or sell my data?

No. TiltCheck is non-custodial and privacy-first. Your bet data stays on your device unless you explicitly choose to sync to the dashboard (encrypted). We never sell data ‚Äî ever.
