What it does

Poker brings structured Hold'em rounds into Discord with clean state handling and anti-chaos moderation controls.

The goal is simple: smooth games, transparent actions, and fewer disputes at the table.

Planned features

  • Action summaries per street
  • Player stats and win rates
  • Hand history archive
  • Session leaderboard

Game Flow

1

Create table

Command spawns a managed game context.

2

Deal and preflop

Players receive hole cards and actions rotate.

3

Bet rounds

Flop, turn, and river run with structured options.

4

Showdown

Best hand is auto-evaluated and pot is distributed.
