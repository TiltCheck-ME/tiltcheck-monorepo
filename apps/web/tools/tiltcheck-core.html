Beta

What is it?

Reality Check Bot monitors pacing signals (messages, commands, high-risk link interactions) and offers gentle cooldown nudges when patterns skew into burnout territory.

No lectures. No policing. Opt-in accountability you can tune to your tolerance.

Key Features

  • Session pacing model
  • Cooldown suggestion nudges
  • Streak & relapse tracking (private)
  • Accountability buddy opt-in
  • High-risk trigger detection (casino spam, loss-chasing)
  • User-tunable thresholds
