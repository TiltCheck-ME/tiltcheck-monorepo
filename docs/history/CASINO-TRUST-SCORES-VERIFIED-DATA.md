## Methodology

### Scoring Formula

Each casino is scored on 5 key dimensions (0-100 each):

**1. Fairness (20% weight)**
- RTP accuracy vs. claims
- License validity
- Audit frequency
- Dispute resolution rate
- Game certification

**2. Support (20% weight)**
- Response time (live chat, email, phone)
- Trustpilot rating
- Complaint resolution speed
- Availability (24/7, business hours)
- Professional quality

**3. Payouts (20% weight)**
- Withdrawal speed (hours/days)
- Success rate (%)
- Processing fees
- Maximum limits
- Fraud incidence rate

**4. Compliance (20% weight)**
- Valid license status
- Regulatory history
- KYC/AML implementation
- Responsible gambling tools
- Data protection (GDPR)

**5. Bonus Quality (20% weight)**
- Offer attractiveness
- Wagering requirement fairness
- Game restriction flexibility
- Bonus expiry period
- Withdrawal liquidity

### Overall Score Calculation

```
Overall Trust Score = 
  (Fairness × 0.20) +
  (Support × 0.20) +
  (Payouts × 0.20) +
  (Compliance × 0.20) +
  (Bonus × 0.20)
```

### Score Interpretation

| Range | Interpretation | Recommendation |
|-------|---|---|
| 85-100 | Excellent | Highly Recommended |
| 75-84 | Above Average | Recommended |
| 65-74 | Average | Acceptable |
| 50-64 | Below Average | Proceed with Caution |
| <50 | Poor | Not Recommended |
