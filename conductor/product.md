# Product Guide: TiltCheck Ecosystem

## 1. Vision

TiltCheck is a modular, AI-assisted ecosystem designed to fix the worst parts of online casino culture — scams, unfair bonuses, predatory patterns, tilt-driven decisions, and chaotic Discord communities.

It doesn't try to stop degens from being degens. It just gives them a smarter, safer, and more transparent way to play.

## 2. Target Audience

The primary audience for TiltCheck is **Discord Degens**: experienced gamblers who are active and influential within online casino communities. These users are technically savvy, skeptical of new platforms, and value tools that provide a clear, immediate edge without taking custody of their funds.

## 3. Core Problem

The most important problem TiltCheck solves is **Tilt Management**. Degenerate gamblers often make poor decisions driven by emotion ("tilt"), leading to significant losses. TiltCheck provides real-time, data-driven feedback and accountability tools to help users recognize these patterns and make smarter choices. While scam protection and community health are important secondary goals, the core focus is on empowering the individual user to manage their own behavior.

## 4. Key Features (MVP)

The Minimum Viable Product (MVP) will focus on delivering immediate value within the user's primary environment: Discord.

*   **JustTheTip Bot**: A Discord bot that provides non-custodial tipping, airdrops, and swaps. This builds trust and utility by offering a core community function without requiring users to deposit funds into a new system. It serves as the primary entry point for user engagement.
*   **Degen-Trust Dashboard**: A web-based dashboard that provides users with insights into their gambling patterns, tilt level, and community trust score. This is the "single source of truth" for a user's status within the TiltCheck ecosystem, providing actionable data and visualizations.

While the SusLink extension is a valuable tool, the MVP will prioritize the bot and dashboard to build a strong user base and data foundation within Discord first.
